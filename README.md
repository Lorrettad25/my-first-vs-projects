# Coding Club

## Description
The Coding Club website is a landing page for parents and children looking for a fun coding club for kids aged 8 to 18. The Coding club offers activities for kids to learn how to code from a young age.

## Features
1. Navigation
2. About us section
3. Sign up form
4. Contact section
